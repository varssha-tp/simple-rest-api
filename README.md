# Simple REST API with FastAPI

A simple RESTful API built with **FastAPI** and deployed on **Render**.

---

## Live Demo

- Base URL: https://simple-rest-api-ior0.onrender.com  
- Get all items: https://simple-rest-api-ior0.onrender.com/items  
- API Docs: https://simple-rest-api-ior0.onrender.com/docs  

---

## What It Does

This project demonstrates a basic REST API with the following endpoints:

- `GET /` → Check if API is running  
- `GET /items` → Retrieve all items  
- `GET /items/{item_id}` → Retrieve a specific item  
- `POST /items` → Add a new item
